Quickstart
----------

``pip``
*******

.. code-block:: console

    pip install ploomber


``conda``
*********

.. code-block:: console

    conda install ploomber -c conda-forge


Learn Ploomber in 2 minutes
***************************

Download example:

.. code-block:: console

    pip install ploomber --upgrade

    python -m ploomber.onboard
